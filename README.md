**Student Grade Calculator**
A simple and user-friendly Python program that calculates a student’s grade based on their marks.
You can enter marks for different subjects, and the program will compute the total, percentage, and final grade.

Features:
*Enter marks for multiple subjects
*Automatically calculates total and percentage
*Displays grade based on percentage
*Beginner-friendly and easy to modify

How to Run:
1.Make sure Python is installed on your system
2.Open a terminal/command prompt
3.Run the following command->
      python grade_calculator.py
    (Replace the filename with your actual Python file name.)

Grade Criteria (Example)
| Average Marks | Grade |
|---------------|-------|
| 95–100        | S     |
| 85–94         | A     |
| 75–84         | B     |
| 65–74         | C     |
| 55–64         | D     |
| 36–54         | E     |
| Below 36      | Fail  |
